# Digital Forensics Lab

Laboratório de Perícia Digital desenvolvido para estudos em Cibersegurança e Computação Forense.

## Objetivos

- Aprender metodologias de investigação digital
- Aplicar cadeia de custódia
- Preservar evidências digitais
- Produzir relatórios periciais
- Simular investigações reais

## Ferramentas

- Kali Linux
- Autopsy
- FTK Imager
- Volatility
- Wireshark

## Sobre o Projeto

Este laboratório foi criado para documentar estudos práticos de Computação Forense e Resposta a Incidentes, simulando cenários reais de investigação digital.

Cada caso contém:
- Descrição do incidente
- Linha do tempo dos eventos
- Cadeia de custódia
- Evidências coletadas
- Relatório técnico
